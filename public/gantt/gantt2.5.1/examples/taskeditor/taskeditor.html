NOTE: For this example to work you have to run it in a web server context. This is a simple example of the task editing window functionality. To open the task editing window please select the task in the tasks tree panel and click the "Show Editor" button in a toolbar. Or you can get the same result by double clicking corresponding task bar in the gantt chart.

Note that the js for the example code is not minified so it is readable. See taskeditor.js.
